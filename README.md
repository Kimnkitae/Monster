# Monster Game
Простая игра, созданная с использованием библиотеки kaplay.

О проекте
Эта игра демонстрирует возможности библиотеки kaplay для создания 2D-игр.

# Как играть
Вы можете запустить игру прямо в браузере по ссылке: Играть в Monster(https://kimnkitae.github.io/Monster/)

# Технологии
• kaplay — основная библиотека.
HTML, JS

• GitHub Pages — хостинг для игры.
